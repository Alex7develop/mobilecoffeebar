# НЕО — Архитектура кофейного опыта

Сайт для премиум выездных кофейных мероприятий. Реализован на **React**, **TypeScript** и **Redux**.

## Стек

- **Vite** — сборка
- **React 19** + **TypeScript**
- **Redux Toolkit** — состояние (язык, интро, аккордеоны, калькулятор)
- **Framer Motion** — анимации (Hero)
- Шрифты: Cormorant Garamond, Montserrat

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`

## Изображения

Картинки для галереи кладите в папку **`public/pages/`** с именами:

- `1.jpg` … `5.jpg` — для блока «Атмосфера» (по порядку с GALLERY_ITEMS).

Если файлов нет, в галерее показываются плейсхолдеры с подписями (V60, Бар на событии, Матча, НЕО, Процесс).

## Структура

- `src/store/` — Redux store и слайсы (ui, calc)
- `src/components/` — секции страницы (Hero, Manifesto, About, Menu, Calculator, Packages, Gallery, Contact, Footer)
- `src/app/constants.ts` — тексты, пункты меню, пакеты, слои кружки

## Особенности

- Интро с «скрембл»-эффектом текста
- Кастомный курсор и фоновый canvas с линиями
- Адаптивная сетка, аккордеоны (Основатели, Меню, Калькулятор)
- Калькулятор стоимости с выбором молока и опций
- Интерактивная SVG-кружка «Напиток события» с подсказками по зонам
